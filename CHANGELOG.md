# Changelog

## Unreleased

## v0.8

- Renamed **Cell Architecture Studio → Scale Explorer** (brand, titles, manual, error banners, i18n) to match the new atom-to-galaxy scope.
- Added the **Cosmos** scale: the Sun, all 8 planets, the Moon, and the Milky Way + Andromeda galaxies.
- Added 5 new categories (Atoms, Stars, Planets, Moons, Galaxies) and a Cosmos branch in the taxonomy tree; **39 specimens across 14 categories**.
- New procedural builders `buildStar`, `buildPlanet`, `buildSpiralGalaxy` (real barred-spiral geometry; point-cloud star discs; rings, cloud bands, atmospheres, orbiting Moon).
- Cosmos specimens placed in a dedicated elevated Atlas band so macroscopic bodies never collide with microscopic biology.
- Live animations for stars (rotation + corona pulse + flares), planets/moons (spin + satellite orbit), and galaxies (disc + core rotation).
- `setOpacity` now fades `THREE.Points`, so the galaxy disc dims correctly with component focus / Read View.
- Added build123d CAD sources for all cosmic bodies (galaxies export GLB point clouds).
- Earlier in this cycle: Hydrogen Atom (atom scale), OSS readiness docs, validation + smoke-test scripts.
